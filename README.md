# WMP-XP-Visualizer
Es un reproductor de audio web que permite cargar y reproducir archivos de música desde tu dispositivo. Incluye una lista de reproducción para gestionar las canciones y controles básicos para reproducir, pausar, detener y cambiar de pista. Además, cuenta con un visualizador gráfico de barras que muestra el espectro de la música en tiempo real y se adapta al tamaño del área de visualización.
